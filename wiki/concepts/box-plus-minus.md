# Box Plus/Minus

BPM is a basketball metric created by Daniel Myers that estimates player value from box score statistics via regression against long-term RAPM. It was a direct inspiration for the box component of [[RAPTOR]], which extends the same philosophy by adding player tracking and play-by-play data.
