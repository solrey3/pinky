# qmd

Local search engine for markdown files with hybrid BM25/vector search and LLM re-ranking, all on-device. Offers both a CLI interface and an MCP server for LLM integration. Recommended by Karpathy as a scaling tool for LLM wikis that outgrow a simple `index.md`.
