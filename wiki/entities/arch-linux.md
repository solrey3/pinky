# Arch Linux

Rolling-release Linux distribution known for minimalism, user control, and the Arch User Repository (AUR). In the newsletter wiki, Arch Linux appears as a supply-chain security surface: the same community packaging model that makes the AUR powerful also creates trust and provenance risk when malicious packages appear.
